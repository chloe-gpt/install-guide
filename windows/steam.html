Chloe's ToUR Install Guide for Windows [Steam]

What You'll Need

- Among Us on Steam

Installation and Downgrading

To start, install Among Us through Steam. If downgrading is required, you can do so by clicking "Properties", then "Betas", then selecting "public-previous". Once you've installed the game, navigate to its install folder - "Manage", then "Browse local files". Duplicate the folder that Among Us is installed to, rename it to something like "Among Us (ToU)", and proceed to the next step.

Mod Installation

Download the "ToU..zip" file from this repository. Extract the contents of the archive into your "Among Us (ToU)" folder - your "Among Us.exe" should be in the same directory as the "BepInEx" folder.

You're Done!

If all has gone well, launching the .exe file in the "Among Us (ToU)" folder should open Among Us with the Town of Us logo on the home screen. Happy tasking!

If your game crashes, doesn't load mods, or another issue occurs, join our Discord and make a ticket in the "#tour-support" channel for assistance.
