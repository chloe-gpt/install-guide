Chloe's ToUR Install Guide for Windows [Itch]

What You'll Need

- Among Us on itch.io (and itch.io app)

Installation

To start, install Among Us in the itch.io app. If downgrading is required, go to your Library, right-click Among Us, then "Manage", then "Switch to another version...". Select the correct version, currently "version 103".

Mod Installation

Right-click on Among Us again, then "Manage", then "Open folder in explorer". Download the "ToU..zip" file from this repository. Extract the contents of the archive into your game folder - "Among Us.exe" and the "BepInEx" folder should both be visible in the same place.

You're Done!

If all has gone well, launching "Among Us.exe" should open Among Us with the ToU logo on the home screen. Happy tasking!

If your game crashes, doesn't load mods, or another issue occurs, join our Discord and make a ticket in the "#tour-support" channel for assistance.
