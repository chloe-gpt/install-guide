Chloe's ToUR Install Guide for Windows [Itch]

What You'll Need

- Among Us on itch.io

Installation

To start, download the correct Among Us version from itch.io. If latest version isn't supported, make sure to download the previous release version.

Mod Installation

Downlod the "ToU..zip" file from this repository. Extract the contents of the archive into your game folder - your "Among Us.exe" should be in the same directory as the "BepInEx" folder.

You're Done!

If all has gone well, launching "Among Us.exe" should open Among Us with the ToU logo on the home screen. Happy tasking!

If your game crashes, doesn't load mods, or another issue occurs, join our Discord and make a ticket in the "#tour-support" channel for assistance.
