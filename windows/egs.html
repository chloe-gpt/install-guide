Chloe's ToUR Install Guide for Windows [Epic]

What You'll Need

- Among Us on Epic Games

Latest Version

ONLY IF LATEST VERSION IS SUPPORTED

To start, install Among Us through Epic Games. Find your game's install directory and duplicate the game folder. Rename it to something like "Among Us (ToU)". Download this zip file, unzip it, and place the exe file inside your copied folder. Proceed to "Mod Installation".

Downgrade

ONLY IF LATEST VERSION IS UNSUPPORTED

Make sure Epic Games is open and you're signed in. Open the PowerShell app and run the command:

iwr https://github.com/whichtwix/EpicGamesDowngrader/releases/latest/download/DowngradeEpic.ps1 -UseBasicParsing | iex

You may have to press "Y" a few times through the prompts on screen. Once it has finished, your downgraded folder will be at "C:\Users\[YOUR NAME]\Games\AmongUs". Proceed to "Mod Installation".

Mod Installation

Download the "ToU..zip" file from this repository. Extract the contents of the archive into your game folder - your "Among Us.exe" should be in the same directory as the "BepInEx" folder.

You're Done!

If all has gone well, launching "EpicGamesStarter.exe" in the "Among Us (ToU)" folder should open Among Us with the ToU logo on the home screen. Happy tasking!

If your game crashes, doesn't load mods, or another issue occurs, join our Discord and make a ticket in the "#tour-support" channel for assistance.
